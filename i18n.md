# 🌍 Guia de Internacionalização (i18n)

## Visão Geral

Este guia explica como usar o sistema de tradução do projeto Anti-Byte.

---

## Idiomas Suportados

| Idioma | Código | Padrão |
|--------|--------|--------|
| Português (Brasil) | pt-BR | ✅ Default |
| Inglês (EUA) | en-US | ✅ Fallback |

---

## Estrutura de Arquivos

```
src/
├── config/
│   └── i18n.ts                    # Configuração central
├── components/
│   └── shared/
│       └── LanguageSelector/      # Componente UI
│           ├── index.tsx
│           └── index.ts
├── locales/
│   ├── pt-BR/translation.json     # Traduções PT-BR
│   └── en-US/translation.json     # Traduções EN-US
└── types/
    └── i18n.ts                    # Type definitions
```

---

## Uso Básico

### Importar e Usar

```tsx
import { useTranslation } from 'react-i18next';

function MeuComponente() {
  const { t } = useTranslation();

  return (
    <h1>{t('hero.title')}</h1>
  );
}
```

---

## Adicionando Traduções

### Passo 1: Criar String no JSON

**pt-BR/translation.json**:
```json
{
  "myNewKey": "Meu novo texto em português"
}
```

**en-US/translation.json**:
```json
{
  "myNewKey": "My new text in English"
}
```

---

## Troubleshooting

### Problema: Texto não traduzido

**Solução**: Verificar se a key existe no JSON de tradução

### Problema: Erro de tipo TypeScript

**Solução**: Adicionar o type ao `TranslationNamespace`
